🧩 What is a Claude Skill?

A Claude Skill is a small, reusable instruction pack — a folder with a SKILL.md file (plus optional reference files) — that teaches Claude a specific way of working. Once you upload one to Claude, Claude automatically reaches for it whenever a conversation matches what it's built for, without you having to re-explain your preferences every single time.

Below is a skill Arsalan personally built and uses. It's free to download — grab the .skill file and upload it straight into Claude.

Free-First Builder

free-first-builder.skill
Download .skill

This skill makes Claude default to free, no-signup, or generous-free-tier tools before ever suggesting a paid one — for literally any build request: coding a project, generating images or video, hosting a backend, building a mobile app, or finding business leads. It's modeled on how Arsalan actually works — cloud-only, budget-conscious, and allergic to a side project quietly turning into a recurring bill. If a task genuinely has no free path, the skill makes Claude say so plainly instead of slipping a paid tool past you.

"build me an app" "how do I host this" "free image generator" "what tool for X" "automate this for free"
✅ Advantages
✔

Free-first by default. Claude checks for a free or no-signup option before ever reaching for a paid tool — even if you never say the word "free."

✔

Names the real catch. No vague "there are limits" — Claude states the exact rate limit, spin-down delay, watermark, or expiring credit.

✔

Verified tool list built in. Ships with a personally-tested reference file covering hosting, mobile builds, AI/LLM access, image/media tools, and more.

✔

Honest "no free option" warning. When a task genuinely needs a paid resource, Claude says so upfront, gives a rough cost sense, and offers a scoped-down free alternative.

✔

Searches instead of guessing. For anything outside the built-in list, Claude web-searches for the current best free option rather than relying on stale training data.

✔

Breaks multi-part builds down. A project like "build me a chatbot" gets split piece by piece — frontend, hosting, LLM calls — so one paid piece doesn't make the whole thing look paid.

📥 How to use it
  1. Download the free-first-builder.skill file using the button above — it's a small zip package, no installation needed.
  2. Open Claude.ai (or the Claude app) and go to Settings → Capabilities → Skills (or the Skills section of your workspace/project settings).
  3. Click Upload skill and select the downloaded .skill file.
  4. Start a new chat and ask a normal build question — e.g. "help me host a small backend for a side project". Claude will automatically pull in the skill and lead with free options.
  5. No need to mention the skill by name — it triggers automatically whenever your request matches what it's built for.
Note: This skill ships with a bundled reference file of personally-tested free tools (Codespaces, Render, Vercel, Codemagic, Google AI Studio, Squoosh, Haikei, ImagineArt, Google Places API, Wappalyzer, and more) — Claude checks that list first, then searches the web for anything not covered.
